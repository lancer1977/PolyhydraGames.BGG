# PolyhydraGames.BGG

BoardGameGeek XML API integration.

## Release

- GitHub Packages is the default package host.
- Use `./pub.sh` to restore, test, pack, and publish.
- Publish to nuget.org only when you explicitly want a public release.

## Contents

- `PolyhydraGames.BGG` - main client
- `PolyhydraGames.BGG.Data.Dapper` - Dapper persistence helpers
- `PolyhydraGames.BGG.Test` - tests
